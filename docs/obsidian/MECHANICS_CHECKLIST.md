# 📋 Чек-лист механик TGPerekup

Этот документ — актуальный статус реализации всех игровых механик.

## 🟢 1. Базовый цикл (Core Loop) — ГОТОВО
- [x] **Передвижение**: Бросок D6, анимация на карте, обработка круга (индекс % 12).
- [x] **Тактический ход**: Выбор клетки (+1, +2, +3) за 1 ед. энергии.
- [x] **Покупка авто**: Списание денег на сервере, добавление в гараж, синхронизация рынка.
- [x] **Продажа авто**: Расчет цены с учетом дефектов и тира, начисление денег на сервере.
- [x] **Энергия**: Регенерация каждые 2 хода, покупка на клетках `sale`.

## 🟢 2. Экономика и Обслуживание — ГОТОВО
- [x] **Новости рынка**: Ротация каждые 5 ходов, влияние на цены всех авто.
- [x] **Ремонт**: Серверная валидация, списание денег, восстановление здоровья авто.
- [x] **Диагностика**: Раскрытие скрытых дефектов за плату.
- [x] **Прокат (Rent)**: Пассивный доход в начале хода владельца.
- [x] **Налоги**: Автоматическое списание за владение авто при каждом броске кубика.

## 🟢 3. Мультиплеер и Взаимодействие — ГОТОВО
- [x] **P2P Долги**:
    - [x] Биржа сделок (DealsView).
    - [x] Займы под залог авто с блокировкой (`isLocked`).
    - [x] **Авто-конфискация**: Переход машины кредитору при просрочке (реализовано на сервере).
- [x] **Гонки (Race)**:
    - [x] Система вызова и ставок.
    - [x] Общий заезд с расчетом бонусов тира.
- [x] **Победа**:
    - [x] Авторитарная проверка баланса на сервере.
    - [x] Событие `victory` и блокировка игры.

## 🟡 4. Соло-режим и Контент — В РАЗРАБОТКЕ
- [x] **База данных**: 30+ моделей авто, 20+ типов дефектов.
- [ ] **Бот Боря (Smart)**: Умное поведение, перехват выгодных лотов (в процессе).
- [ ] **Квесты**: Сюжетные задания на подбор конкретных авто.
- [ ] **Репутация**: Влияние на скидки в сервисах и удачу.

## 🛠 Технические стандарты
1.  **Zod DTOs**: Все данные валидируются схемами из `shared/dtos`.
2.  **Decimal.js**: Финансовая точность во всех расчетах.
3.  **Authoritative Server**: Критическая логика вынесена в `packages/server`.
