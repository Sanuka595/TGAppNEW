# 📋 Чек-лист механик TGPerekup

Этот документ — "Source of Truth" по текущему прогрессу реализации. Используется для отслеживания багов и планирования фич.

## 🟢 1. Базовый цикл (Core Loop) - ГОТОВО
- [x] **Передвижение**: Бросок D6, анимация на карте.
- [x] **Тактический ход**: Потребление 1 ед. энергии для выбора клетки (1-6).
- [x] **Покупка авто**: Списание денег, добавление в гараж, удаление из маркета.
    - *Нюанс*: Синхронизация рынка (updateMarket) теперь работает.
- [x] **Продажа авто**: Начисление денег, расчет цены с учетом Tier и дефектов.
- [x] **Энергия**: Реген при старте хода, покупка за $500.

## 🟡 2. Экономика и Обслуживание - ЕСТЬ НЮАНСЫ
- [x] **Новости рынка**: Ротация каждые 5 ходов.
    - *Нюанс*: Влияют на `calculateSellPrice` и `calculateCurrentMarketValue`.
- [x] **Ремонт**: Списание денег, восстановление Health.
- [x] **Диагностика**: Раскрытие `isHidden` дефектов за $200.
- [x] **Прокат (Rent)**: Пассивный доход раз в ход.
- [/] **Случайные события (15%)**: ГАИ и Удача при каждом ходе.
    - *Нюанс*: Работают локально у каждого игрока.

## 🟠 3. Мультиплеер и Взаимодействие - В РАЗРАБОТКЕ
- [/] **P2P Долги (Займы)**:
    - [x] Создание оффера и публикация на бирже.
    - [x] Принятие оффера с блокировкой авто (`isLocked`).
    - [ ] **Критично**: Конфискация при просрочке (сейчас заглушка, авто не переходит кредитору).
- [/] **Гонки (Race)**:
    - [x] MVP Соло-гонка против шансов.
    - [ ] **План**: Переделка в дуэль (Вызов -> Ставка -> Принятие -> Общий заезд).
    - [ ] Бонусы за класс машины (Rarity +2, Scrap -1).
- [ ] **Победа**:
    - [x] Проверка на клиенте при продаже.
    - [ ] **План**: Перенос валидации на сервер (Phase 3).

## 🔴 4. Контент и Solo-режим - ПЛАН (Phase 4)
- [ ] **Бот Боря**: Автоматические ходы, перехват выгодных лотов.
- [ ] **Система репутации**: Влияние на цены в сервисе и шанс победы в гонках.
- [ ] **Сюжетные квесты**: "Пригнать машину определенного цвета/марки".
- [ ] **База авто**: Расширение до 50+ моделей (сейчас ~15).

## 🛠 Технические нюансы (Dev Notes)
1.  **Zod DTOs**: Любое изменение в объектах `Car`, `Player`, `RoomState` должно начинаться с `packages/shared/src/dtos/`.
2.  **Haptics**: Интегрированы во все важные действия (покупка, штраф, кости).
3.  **Logs**: Вести с юмором, используя `addLog(text, type)`.
