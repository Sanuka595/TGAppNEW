# 📋 Чек-лист механик TGPerekup

Этот документ — актуальный статус реализации всех игровых механик.

## 🟢 1. Базовый цикл (Core Loop) — ГОТОВО
- [x] **Передвижение**: Бросок D6, анимация на карте, обработка круга (`position % 12`).
- [x] **Тактический ход**: Выбор клетки (+1, +2, +3) за 1 ед. энергии (`MAX_ENERGY = 3`).
- [x] **Случайные встречи**: 15% шанс штрафа ГАИ или денежной находки (`resolveRandomEncounter()`).
- [x] **Покупка авто**: Списание денег на сервере, добавление в гараж, синхронизация рынка.
- [x] **Продажа авто**: Расчет цены с учётом дефектов и тира, начисление на сервере, проверка победы.
- [x] **Энергия**: Покупка за $500 (`ENERGY_PURCHASE_COST`) на клетках `sale`.

## 🟢 2. Экономика и Обслуживание — ГОТОВО
- [x] **Новости рынка**: Ротация каждые 5 ходов, влияние на цены через `applyNewsEffects()`.
- [x] **Ремонт**: Серверная валидация, списание денег, пересчёт `health`.
- [x] **Диагностика (гараж)**: Раскрытие скрытых дефектов за $200 (`DIAGNOSTICS_COST`).
- [x] **Диагностика (рынок)**: Диагностика лота прямо на витрине перед покупкой.
- [x] **Обновление рынка**: Новый лот за $500 (`MARKET_REFRESH_COST`) через `generateMarketForCell()`.
- [x] **Прокат (Rent)**: Пассивный доход по `calculateRentIncome(tier)`.
- [x] **Налоги**: Автоматическое списание при каждом броске через `calculateOwnershipTax()`.

## 🟢 3. Мультиплеер и Взаимодействие — ГОТОВО
- [x] **P2P Долги**:
    - [x] Биржа сделок (`DealsView`).
    - [x] Займы под залог авто с блокировкой (`isLocked`).
    - [x] **Авто-конфискация**: Сервер переводит машину кредитору при `turnsLeft = 0`.
- [x] **Гонки (Race)**:
    - [x] Вызов на дуэль с выбором ставки (≤ 50% баланса).
    - [x] Авторитарный расчёт: D6 + `TIER_RACE_BONUS` из `shared` + бонус инициатора (+1).
- [x] **Победа**: Авторитарная проверка `balance >= winCondition` при каждой продаже.

## 🟡 4. Соло-режим и Контент — В РАЗРАБОТКЕ
- [x] **База данных**: 30+ моделей авто, 20+ типов дефектов.
- [x] **Формула соло-гонки**: `calculateSoloRaceWinChance(car)` в `shared`.
- [ ] **Бот Боря (Smart)**: Умное поведение, перехват выгодных лотов.
- [ ] **Квесты**: Сюжетные задания на подбор конкретных авто.
- [ ] **Репутация**: Влияние на скидки в сервисах и удачу.

## 🛠 Технические стандарты
1. **Zod DTOs**: Все данные валидируются схемами из `shared/dtos`.
2. **Decimal.js**: Финансовая точность во всех расчётах.
3. **Authoritative Server**: Критическая логика — в `packages/server`.
4. **Shared-first**: Формулы и константы — в `packages/shared`, импортируются клиентом и сервером.
5. **Slice pattern**: Zustand `StateCreator` — каждый домен в своём слайсе, `gameStore.ts` — тонкий ассемблер.
